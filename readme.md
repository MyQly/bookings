# Bookings and Reservations

This is the repostory for a bookings and reservations project.

- Built in Go (1.20)
- Depends on:
    - [chi](https://github.com/go-chi/chi) - for routing
    - [nosurf](https://github.com/justinas/nosurf) - for csrf protection
    - [scs](https://github.com/alexedwards/scs) - for session management


# How To Run

1. Pull this repository to a directory of your choice.

2. Run the following in the terminal:

    `go run ./cmd/web/`
